帮助

快速开始

  1. 在项目根目录运行 python serve.py
  2. 浏览器访问 http://127.0.0.1:8000/admin.html
  3. 编辑各项内容，焦点离开输入框后自动保存
  4. 主页刷新即可看到更新（同源下实时同步）

自动保存机制

编辑任何字段后，焦点离开输入框（blur）或上传图片完成时，会自动保存到 projects.json 并同步本机草稿。顶栏右侧显示保存状态。

图片管理

  • 点击「+ 添加图片」打开图片选择器
  • 可从 assets/images/ 选择已有图片
  • 可上传新图片，自动以时间戳命名保存到 assets/images/
  • 首张图片默认作为封面，可点击「设为封面」更改

笔记 Markdown 语法

  • # 标题 / ## 副标题 / ### 小标题
  • **粗体** / *斜体*
  • [链接文字](https://...)
  • `行内代码` / ```代码块```
  • - 无序列表 / 1. 有序列表
  • > 引用 / --- 分割线

同步到 GitHub Pages

  • 方式一（推荐）：本地 python serve.py → 编辑保存 → git add . && git commit && git push
  • 方式二：GitHub 网页打开 assets/data/projects.json → 编辑铅笔 → 粘贴 JSON 后提交
  • 方式三：访问 github.dev 仓库地址，浏览器内 VS Code 编辑提交

注意事项

  • 主页和后台请使用同一地址访问（都用 127.0.0.1:8000 或都用 localhost:8000），否则 localStorage 草稿无法跨 origin 同步
  • 直接用 file:// 协议打开 admin.html 仅能保存草稿到本机，无法写入 projects.json
  • 导出文件仅作为备份手段，正常使用无需手动导出
